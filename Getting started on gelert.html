Getting started on gelert
  1. Create data_config.yml file for your specific project Rocky Data Organization
  1. Create pipeline_config.yml file for your specific project Rocky Data Organization
  1. Set up .bashrc file. See: template .bashrc
  1. In the folder containing your .yml files, create run_project.py with these commands:
  • import CPAC
  • config_file         = "pipeline_config.yml"
  • subject_list_file   = "data_config.yml"
  • CPAC.pipeline.cpac_runner.run(config_file, subject_list_file, "test")
  1. Navigate to gelert and run the python script:
  • python run_project.py
