Several subjects have duplicate ids (need to deal!)
 
The code below will find the duplicates and associated input paths:
  • import CPAC_subject_list
  • import collections
  • sublist = CPAC_subject_list.subjects_list
  • subs = [ x['Subject_id'] for x in sublist ]
  • duplicate_subs = [ x for x,y in collections.Counter(subs).items() if y>1 ]
  • overlaps = { s : [ x for i,x in enumerate(sublist) if x['Subject_id'] == s ] for s in duplicate_subs }
The gist here is that for almost all overlapping subject ids, this won't matter since the names of the scan are different. The issue is only for sub68050 who should not be included in the analyses.
